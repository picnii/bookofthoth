Chapter 1 : Installation

บทความนี้แนะนำวิธีการติดตั้ง WordPress โดยสิ่งที่เราต้องมีก็คือ

1. Webserver

2. WordPress

Webserver  มีได้ 2 ทางคือ

1. ลงเว็บใช้ App Serv จำลองเป็น Webserver ในเครื่องเรา

2. เช่า Host และ Domain จริงๆ ซึ่งสิ่งที่ Web Server  ต้องรองรับก็คือ php และ mysql

ในที่นี้เราจำลอง Host และ Domain มาให้ แต่เราจะทำการติดต่อได้ต้องผ่าน FTP ซึ่งมีหลายตัวให้เราเลือกใช้ แต่ขอยกมาเพียงหนึ่งตัวเพื่อสาธิต ให้ไปที่  https://filezilla-project.org/ ดาวโหลด FTP ชื่อ Filezilla จะมีหน้าตาโปรแกรมดังภาพ

 

2. WordPress

ไปที่ http://wordpress.org เพื่อทำการ download

เมื่อทำการ download เสร็จ เราจะได้ zip file คลายzipออกมาจะได้ folder ชื่อ wordpress กดเข้าไปใน folder เราจะเห็นเหมือนภาพ

ที่นี้เราต้องย้าย file ทั้งหมดในนี้ไปไว้ที่ ftp ของเรา เราต้อง โฮสต์ ชื่อผู้ใช้ รหัสผ่าน

ตัวอย่าง โฮสต์ : 192.168.1.30 ชื่อผู้ใช้ : student_05 รหัสผ่าน : picnii.me  เมื่อ login ได้แล้วให้ย้ายไฟล์ภายใน wordpressไปใน FTP จะได้ดังภาพ

ไปที่ web browser(แนะนำ Chorme)  พิมพ์ 192.168.1.30/student_05 จะเห็นเหมือนในภาพ

กด Create a Configuration File > Let’s Go เมื่อมาถึงหน้าดังภาพ


กรอกข้อมูลตามภาพ แล้วกด Submit  จะมาอีกหน้าต่างหนึ่งแต่จะถูกฟ้องว่า ยังไม่มี file wp-config.php

ให้เราเปิด notepad แล้ว copy ข้อความในแถบฟ้ามาวางใน notepad แล้วsave ตอน save เปลี่ยน .txt เป็น all file ตั้งชื่อไฟล์ว่า wp-config.php แล้วย้ายมาไว้ใน FTP

เมื่อย้ายมาลงใน FTP แล้วกด Run the install เราจะมาที่หน้าต่าง Information ให้เรากรอกข้อมูลให้ครบ

กด install แล้วจะมาที่หน้า http://192.168.1.30/student_05/wp-login.php ที่นี้เราจะ login เข้า wordpress ได้ ถือว่าเป็นอันเสร็จสิ้นในการติดตั้ง Wordpress

Chapter 2 : สร้าง Page

WordPress เป็น CMS ตัวนึงที่ใช้บริหารจัดการเนื้อหา โดยเน้นในความง่ายของการสร้าง และ บริหารจัดการเนื้อหา โดยเนื้อหาหลักๆของ WordPress จะมี 2 ประเภท คือ Post และ Page

Page คือ ส่วนของเนื้อหาที่ไม่ค่อยมีการเปลี่ยนแปลง (Static) เช่นหน้า Contact Us, About me , หรืออื่นๆ

ก่อนอื่นเรามาดูหน้าตาของ Dashboard โดย พิมพ์ว่า localhost/myblog/wp-admin ที่ web browser เพื่อทำการ log in

เมื่อ log in เสร็จ แล้วเราก็จะมาที่หน้า Dashboard

ที่นี่เรามาดูการสร้าง Page

เลือก Add New ที่ ฟอรั่ม Page เพื่อสร้าง page ใหม่ เมื่อเราใส่เนื้อหาเรียบร้อย กด Publish เพื่อเผยแพร่เนื้อหา

กดที่หัวข้อ  All Pages  เพื่อแสดงข้อมูลหน้า Page  เรามาสามารถจัดการแก้ไข เพิ่มเติม และลบ หน้าเพจจากหน้านี้

Chapter 3 : สร้าง Post

Post จะเป็นเนื้อหาที่มี การ Update บ่อยๆ (Dynamic) เช่น บทความต่างๆ, Tips ต่างๆ, เนื้อหา Blog

เราลองคลิ๊กหน้า Posts ที่หน้า Dashboard เราจะเห็นหัวข้อต่างๆ เช่น All Posts, Add New, Categories, Tags

ในส่วนของหน้า All Posts และ Add New จะคล้ายกับในส่วนของ Page แต่ต่างกันเล็กน้อยตรงที่ในส่วนของโพสต์นั้นเราจะสามารถ ใส่ Categories และ Tagsได้

ที่นี้เราจะลองมาดูส่วนที่แตกต่าง คือในส่วนของ Categories Categories คือการจัดหมวดหมู่ในกับ Post ของเรา เราสามารถสร้างหมวดหมู่ขึ้นมาใหม่ได้ตามที่เราต้องการ เช่นหมวดหมู่ของ ร้านอาหาร เป็นต้น

ในส่วนของ Tags คือ การใส่รายละเอียดแบบเฉพาะเจาะจงให้กับโพสต์ เช่น เราสร้างโพสต์ แล้วใส่หมวดหมู่ อาหาร ไว้ แล้วเราติด Tags ร้านอาหาร อาหาร อาหารญี่ปุ่น ซูชิ ซาชิมิ เป็นต้น

Chapter 4 : Theme

Theme คือ หน้าตาของ Webblog ที่นี่เรามาดูวิธีเปลี่ยน Theme ครับ

ก่อนแรกเราไปที่ web browser แล้วไปที่ http://wordpress.org/themes/

เมื่อเราดาวโหลดมาแล้ว นำไฟล์มาเก็บไว้ที่ C:\AppServ\www\myblog\wp-content\themes แล้วไปที่หน้า Dashboard ของเรา แล้วไปที่ Appearance >Themes

ที่นี้เราจะมาอยู่ที่หน้า Manage Themes แล้ว กด Activate เพื่อเปลี่ยน Themes นั้น

Chapter 5 : Plugin

plugin คือ การเพิ่มความสามารถให้ wordpress เช่น ทำให้ wordpress สามารถ search Google ได้ดีขึน / มี Slide show หรืออื่นๆ

โดยเราสามารถเข้าไปค้นหา Plugin ได้ที่  http://wordpress.org/plugins/ โดย Plugin ที่ค้นหาต้องรองรับกับ version ของ Wordpress ที่เราใช้งานอยู่ ซึ่งเราสามารถดู version ของ wordpress เราได้ที่ http://localhost/wordpress/wp-admin/update-core.php

ตัวอย่าง Wordpress น่าลอง

Download Plugn นี้มา ลอง http://wordpress.org/plugins/nextgen-gallery/

จะได้ไฟล์ zip คลาย Zip แล้วนำไปไว้ที่ C:\AppServ\www\myblog\wp-content\plugins

มาที่หน้า Dashboard แล้วไปที่ Plugins  เราจะเห็น Plugins ทั้งหมด แล้วเลือก Plugin ที่เราต้องการเพื่อกด Activate

Chapter 7 : User and Setting

เราสามารถเข้าไปปรับการตั้งค่าต่างๆได้ที่  Setting

ตั้งค่าทั่วๆไป (ชื่อเว็บไซต์ email admin ) / การให้ทุกคนสามารถสมัครเว็บได้หรือไม่?

http://localhost/myblog/wp-admin/options-general.php

วิธีการเขียนต่างๆ (แปลง ข้อความเป็น emoticon,  default category ตั้งต้นของบทความ)

http://localhost/myblog/wp-admin/options-writing.php

วิธีการที่ผู้ใช้งาน อ่านจากเว็บไซต์

  • หน้าแรกแสดงเป็น  post ล่าสุด หรือ page
  • แสดงสูงสุดกี่ post ต่อหน้า

http://localhost/myblog/wp-admin/options-reading.php

วิธีการคุม discuss ในเว็บ

  • ใครสามารถที่จะแสดงความเห็นในโพสต์ได้บ้าง อาจจำกัดแค่คนที่เป็นสมาชิก
  • เตือนเจ้าของเว็บตอนไหนบ้าง เช่น เมื่อมีการแสดงความคิดเห็น
  • comment แสดงเท่าไหร่ต่อ1หน้า เมื่อเกินจะขึ้นหน้าใหม่
  • ต้อง approve comment ก่อนหรือเปล่า

http://localhost/myblog/wp-admin/options-discussion.php

link ตอนแชร์เป็นอย่างไร

http://localhost/myblog/wp-admin/options-permalink.php

เพิ่ม แก้ไข ข้อมุล user ในเว็บ

http://localhost/myblog/wp-admin/users.php
